Important FAQ ?

1. What is the Document Object Model (DOM)?

-The Document Object Model (DOM) is a programming interface for web documents.

2.How do you select an element from the DOM?

-To select an element from the DOM, use methods like getElementById('id') for a single element or querySelector('.class') for the first match. For multiple elements, use querySelectorAll('.class') or getElementsByClassName('class').

3.What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events from its child elements. It leverages event bubbling (where events propagate up the DOM tree) and is useful because it reduces the number of event listeners, improves performance, and dynamically handles elements added later.

4.How do you manipulate an element's attributes and styles using the DOM?

-To manipulate an element's attributes, use setAttribute() to change or add attributes and getAttribute() to retrieve them. For styles, access the style property and modify CSS properties directly.
